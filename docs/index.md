# Bloomberg Terminal Guide

This site is a student-built guide to using the Bloomberg Terminal, with a focus on equity research workflows.

## About This Guide

Originally prepared for the UW Investment Group by Luke Alexander.

Last updated: February 2026

Because this guide was written around a university Bloomberg Terminal setup, some functions or entitlements may differ from a full professional subscription.

> Note: Neither the UW Investment Group nor the authors are affiliated with Bloomberg Inc.

## Current Coverage

The site is currently focused on a streamlined equities path.

- [Equities Overview](equities/index.md)
- [Company Overview](equities/company-overview/index.md)
- [DES - Security Description](equities/company-overview/security-description.md)

## Suggested Starting Point

Start with [DES - Security Description](equities/company-overview/security-description.md) to confirm the security, review the company profile, and get the key market context in one place.

## What's In Scope

- Security identification and classification
- Core company profile information
- Key price, valuation, and dividend context
- Basic corporate and management details
